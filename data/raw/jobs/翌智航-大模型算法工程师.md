# 翌智航 - 大模型算法工程师

## 岗位职责

- 负责大模型算法设计、训练和调优，包括预训练、指令微调和强化学习。
- 基于国产芯片及配套框架，进行大模型高效移植和性能优化。
- 将通用大模型适配到应急、工业等垂直领域，提供解决方案。
- 研究稀疏化、量化、模型蒸馏等轻量化技术。
- 跟踪 AGI 前沿动态。
- 负责多模态融合和智能体 Agent 在复杂任务中的应用开发。

## 任职要求

- 硕士及以上学历，计算机、人工智能等相关专业。
- 精通 PyTorch 或 TensorFlow。
- 熟悉 Transformer、MoE 等模型架构。
- 熟悉大模型分布式训练，例如 DeepSpeed。
- 有 CUDA 或算子优化经验优先。
- 有国产芯片开发经验优先。
- 深入理解大模型训练全流程及前沿技术。
- 具备扎实数据处理和算法基础。

## 加分项

- 完整参与过大模型微调、部署或优化流程。
- 有多模态、Agent 或垂直领域模型落地经验。

## 高频技术

PyTorch、TensorFlow、Transformer、MoE、DeepSpeed、CUDA、国产芯片、量化、蒸馏、稀疏化、多模态、Agent、AGI。

## 我的匹配点

已有 Agent 和大模型应用工程基础。

## 我的差距

需要补大模型训练全流程、分布式训练、量化蒸馏和底层性能优化知识。
