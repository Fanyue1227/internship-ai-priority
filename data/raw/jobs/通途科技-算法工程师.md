# 通途科技 - 算法工程师

## 岗位职责

- 在导师指导下理解业务目标，参与公司数据算法体系的学习和辅助建设。
- 协助完成数据分析、挖掘模型的搭建与验证。
- 学习传统知识图谱构建方法。
- 尝试使用大语言模型 LLM 优化知识图谱构建流程。
- 学习 Neo4j 等图数据库的基本使用，协助完成知识存储、查询和简单图分析任务。
- 学习常用自然语言处理算法，了解 LLM、AIGC 基本原理。
- 参与相关技术验证和实验。
- 养成技术文档习惯，协助撰写算法说明、实验记录和技术文档。

## 任职要求

- 本科或硕士在读，计算机、人工智能、软件工程、通信工程、自动化、应用数学等相关专业。
- 具备良好数学基础，例如线性代数、概率论、最优化方法。
- 了解常用机器学习和数据挖掘算法，例如分类、聚类、回归。
- 了解主流深度学习工具，如 TensorFlow、PyTorch、MXNet 等。
- 对新技术有好奇心，具备主动学习能力和团队协作意识。

## 加分项

- 有数学建模竞赛经历。
- 有 NLP、知识图谱、LLM 相关课程作业或个人项目经验。
- 有 GitHub 仓库、技术笔记、博客、Notion 或 GitBook 记录。
- 能保证连续 3 个月以上实习时间。

## 高频技术

Python、知识图谱、Neo4j、LLM、AIGC、NLP、PyTorch、TensorFlow、MXNet、机器学习、数据挖掘、技术文档。

## 我的匹配点

数学背景、建模竞赛、RAG 技术文档问答和 Agent 项目比较匹配。

## 我的差距

需要补知识图谱、Neo4j、机器学习基础模型和图数据库查询。
